# ESEIMS

ESEIMS is a project thought for system administrators that creates automatically three Azure VMs (representing two desktop pcs and a server), configures them, through Ansible, with monitoring tools (Telegraf, Kapacitor, Grafana, InfluxDB) and runs on the server a web application (Angular, NodeJS, Express, MySQL) that contains dashboards showing real-time data with performance stats of all the VMs, alarms management, users registration and manual ticket creation for users.
